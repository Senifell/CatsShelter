Сайт приюта для котов.

Этот проект представляет собой pet-project (CSS + HTML + JS). Создан для тренировки навыков верстки и программирования. 
На данный момент закончен. Возможны правки в будущем по мере улучшения навыков. 

Запуск:
 - node app.js

Реализованные фишки:
 - Слайдер фотографий
 - Разворот меню на маленьких экранах
 - Модальные окна
 - Мини-анимация
 - Копирование значений в буфер обмена с соответствующим сообщением
 - Фильтрация данных
 - Карта (API Яндекс карты)
 
 Данные хранятся в JSON-файле.
 
Скриншоты приложения для быстрого просмотра представлены в папке image-site-for-fast-look
 
В проекте используются сгенерированные ИИ изображения.
